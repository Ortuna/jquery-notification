jquery-notification

Demos

  • Default 
    $.createNotification({ content: 'Hola mundo!' })
  • Lower left 1 second 
    $.createNotification({
	horizontal: 'left',
	vertical: 'bottom',
	content: 'Hey hey Mar',
	duration: 1000
})
  • Image template 
    $.createNotification({
	content: '<img src="https://twimg0-a.akamaihd.net/profile_images/770845756/45_normal.jpg"> Hi, I\'m GG Allin'
})
  • Slow global fadeIn effect 
    $.extend($.notificationOptions, {
	fadeIn: 2000
})
$.createNotification({
	content: 'fadeIn is now set to 2000ms'
})
  • Reset global fadeIn effect 
    $.extend($.notificationOptions, {
	fadeIn: 400
})
$.createNotification({
	content: 'fadeIn is now set to 400ms'
})
  • Limit 2 (hit me many times!) 
    $.createNotification({
	content: 'Ho ho!',
	limit: 2,
	vertical: 'bottom'
})
  • Limit 2 with queue (hit me many times!) 
    $.createNotification({
	content: 'ba dum ts!',
	limit: 2,
	queue: true,
	horizontal: 'left'
})
  • Sticky 
    $.createNotification({
	content: 'I\'m not leaving punk',
	duration: null
})
  • Sticky, close on click 
    $.createNotification({
	content: 'I may leave, if you click on me',
	duration: null,
	click: function() {
		this.hide();
	}
})

Installation

Include script after jQuery

<script src="jquery.notification.js"></script>

Include stylesheet

<link rel="stylesheet" href="notification.css">

Copy images (using same path of stylesheet)

Usage

$.createNotification(options)

Options

click

Click callback

content

Notification content

duration

On screen duration in milliseconds, set it to 0 for sticky, default 5000

fadeIn

Fade in effect duration in milliseconds, default 400

fadeOut

Fade out effect duration in milliseconds, default 400

limit

Visible limit per board, when reached notifications wont be created, default false

queue

If queue is active and limit is reached notifications will be shown when other hides, default false

slideUp

Slide up effect duration in milliseconds, default 200

horizontal

Horizontal alignment, default right

vertical

Vertical alignment, default top
